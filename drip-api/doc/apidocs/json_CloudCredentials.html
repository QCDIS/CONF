CloudCredentials Data Type

This class represents the cloud credentials for a cloud. They are used by the provisoner to request for resources.

Properties
name data type description
id string
key string The key for the cloud provider.
keyIdAlias string The key id for the cloud provider or the key alias.
loginKeys array of LoginKey The login keys
cloudProviderName string The name of the cloud provider
keypass string The password for key stores
Properties inherited from OwnedObject
owner string
